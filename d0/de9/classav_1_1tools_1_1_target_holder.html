Detailed Description

TargetHolder base class

TargetHolders are created by or passed through Selectors and passed to Tools. They typically hold an av::FieldContainer reference that should be manipulated by a tool.

SFContainer av::tools::TargetHolder::Creator

May be used to define the creator of this TargetHolder instance, typically a Selector.

SFBool av::tools::TargetHolder::Keep

If false, this TargetHolder may be replaced with another one, which is default. If true, it is kept as a parent from the new TargetHolder.

MFTargetHolder av::tools::TargetHolder::ParentTargets

Selectors put incoming TargetHolders here, if they want to replace the Targetholder with a new one and the Keep value is true.

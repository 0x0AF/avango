((expr)                                                               \
   ? static_cast<void>(0)                                               \
   : (::av::assert_fail(#expr, __FILE__, __LINE__, AV_ASSERT_FUNCTION), \
      static_cast<void>(0)))

Improved assert. It tries to start a debugger when an assertion occurs. The default debugger (gdb) can be overridden by setting the AV_DEBUGGER environment variable accordingly.

Try to get a nicely formatted function name from the current context. AV_ASSERT_FUNCTION tries to use the default __ASSERT_FUNCTION from GLIBC.

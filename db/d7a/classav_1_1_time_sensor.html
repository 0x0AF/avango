Detailed Description

Simple time sensor. The reference time is initialized with the current real time on instantiation but may be set to a user-defined value.

RealTime is an adaptor for gettimeofday() always containg the current time and is touched in evaluate().

Time is updated in evaluate().

This function is called whenever one of the fields has been changed. You want to overload this function, if you want your node to respond to field changes in a non standard way. evaluate() is called only once per simulation frame, even if multiple fields have been changed.

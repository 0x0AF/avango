Detailed Description

An Avango NG device to communicate with any devices that are registered via Linux event system (i.e. /dev/input/something using linux/input.h) such as mice, keyboards, joysticks and gamepads.

Implementation of Avango's HIDInput on Windows Provides full support for mice and keyboards; however, other HID devices might work incorrectly. There is also no support for LEDs.

Detailed Description

template<typename T>
class av::Link< T >

A typed smart pointer class. The Link class template makes the use of smart pointers typesafe with respect to the C++ compiler. T must be a class which is derived from Base.

template<typename T>
T* av::Link< T >::operator-> ( ) const [inline]

Dereferences a link like the standard '->' operator does on a normal pointer. If the link is empty or not valid an exeption would be thrown, if the dumb SGI C++ compiler would support exeption handling. For now an assertion is raised, i.e. the program is terminated.

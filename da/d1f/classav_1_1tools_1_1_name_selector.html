SFBool av::tools::NameSelector::TreatAsRegularExpression

Specify, if the given strings in SelectableNames and SelectableTargetNames should be treated as regular expressions. Default value: False. If EqualNameOnly and TreatAsRegularExpression are both set to True, each regular expression is surrounded by .*<Expression>.*

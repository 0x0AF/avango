Detailed Description

This class traverses the scenegraph under RootNode and accumulates the transformation matrix. It then lets the SoundRenderers stored in Renderers update their LocalSources stored in a SoundSource that is found in the tree.

void av::sound::SoundTraverser::evaluate ( void  ) [virtual]

This function is called whenever one of the fields has been changed. You want to overload this function, if you want your node to respond to field changes in a non standard way. evaluate() is called only once per simulation frame, even if multiple fields have been changed.

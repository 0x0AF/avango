void av::tools::FieldSelector::evaluate ( void  ) [virtual]

This function is called whenever one of the fields has been changed. You want to overload this function, if you want your node to respond to field changes in a non standard way. evaluate() is called only once per simulation frame, even if multiple fields have been changed.

Member Data Documentation

SFBool av::tools::FieldSelector::EqualNameOnly

Set to false if also objects should be selectable, which contain the given names as substring. Otherwise only equal names are selectable, which is the default.

SFBool av::tools::FieldSelector::SearchOSGNodePaths

Set to true if the selectable names should also be searched in the OSG node paths.

SFBool av::tools::FieldSelector::SearchTargetHolderNodePaths

Set to true if the selectable names should also be searched in the OSG node paths found in NodePathTargetHolder objects.

SFString av::tools::FieldSelector::SelectableFieldName

Defines the name which is used to select targets. The output is in SelectedTargets.

MFTargetHolder av::tools::FieldSelector::Targets

Defines the input targets from which the named objects are selected.

SFBool av::tools::FieldSelector::TreatAsRegularExpression

Specify, if the given strings in SelectableNames and SelectableTargetNames should be treated as regular expressions. Default value: False. If EqualNameOnly and TreatAsRegularExpression are both set to True, each regular expression is surrounded by .*<Expression>.*

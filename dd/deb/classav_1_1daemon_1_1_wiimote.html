Detailed Description

An Avango NG device for communication with a Nintendo Wiimote control. To use this device it is necessary to install the cwiid library. (http://abstrakraft.org/cwiid/ under GPL license) with patches #34 #35 and #36 (see tickets under cwiid webhome). In conjunction with a running wminput daemon started like this: "wminput [ma:ca:dd:re:ss:01] &" the Linux event system will propagate the inputs via /dev/input/.

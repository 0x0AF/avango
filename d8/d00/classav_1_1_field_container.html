Detailed Description

The FieldContainer class is the base class for all classes which contain fields (see Field) to encapsulate object attributes. It provides methods to query number, name, type and value of any field it contains.

It provides a complex macro package which should be used by derived classes in order for runtime typing, reference counting, field declaration and inheritance to work.
